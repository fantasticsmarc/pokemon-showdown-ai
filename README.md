# Pokemon Showdown AI

`config.json` for ladder play can use one shared Showdown account for the selected bot:

```json
{
  "ladder_account": {
    "username": "your_showdown_account",
    "password": "your_showdown_password"
  }
}
```

`main.py` only creates the ladder bot you choose, so one ladder account is enough if you play one bot at a time.
